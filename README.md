# 🔗 MakeItShort – URL Shortener

A clean and modern full-stack URL shortener that converts long links into short, shareable URLs with QR code generation.

---

## 🚀 Features

* 🔗 Shorten long URLs instantly
* 📋 Copy to clipboard
* 📱 Generate QR codes for each link
* ⚡ Fast backend with MongoDB
* 🎨 Clean, modern UI

---

## 🛠️ Tech Stack

**Frontend**

* React (Vite)
* Axios
* QR Code Generator

**Backend**

* Node.js
* Express
* MongoDB (Mongoose)

---

## ⚙️ Environment Variables

### Backend (`.env`)

```
MONGO_URI=your_mongodb_connection
BASE_URL=your_backend_url
FRONTEND_URL=your_frontend_url
```

### Frontend (`.env`)

```
VITE_BACKEND_URL=your_backend_api_url
```

---

## 📦 Installation

### 1. Clone the repo

```
git clone https://github.com/azmil666/url-shortener.git
```

### 2. Backend setup

```
cd backend
npm install
npm run dev
```

### 3. Frontend setup

```
cd frontend
npm install
npm run dev
```
